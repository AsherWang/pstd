## Project Standard Advisor 
一个命令行工具，用来给js项目添加一些配置或者代码检查等

## 功能
- 创建文件 `.npmrc`
- 添加 `eslint` 支持，规范代码 (实际上是安装 `eslint` 然后运行 `eslint --init`)
- 添加 `commitlint`，规范提交
- 自动检查并跳过已经添加的

## 安装
`npm install pstd -g`

## 运行
在项目根目录执行 `pstd`
